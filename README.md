#  Sistema de Gestión de Inventario

Este proyecto es un sistema de gestión de inventario desarrollado en Python. Permite administrar productos mediante un menú interactivo en consola.

---

##  Funcionalidades

* Agregar productos
* Mostrar inventario
* Buscar productos
* Actualizar datos
* Eliminar productos
* Ver estadísticas
* Guardar en archivo CSV
* Cargar desde archivo CSV

---

##  Estructura del proyecto

El proyecto está dividido en 3 archivos principales:

* **main.py** → Contiene el menú principal y la interacción con el usuario
* **funciones.py** → Contiene la lógica del sistema (agregar, buscar, eliminar, etc.)
* **archivos.py** → Manejo de archivos CSV (guardar y cargar datos)

---

##  Cómo ejecutar

1. Asegúrate de tener Python instalado
2. Ejecuta el archivo principal:

```bash
python main.py
```

---

##  Archivos

El sistema guarda los datos en:

* `inventario_final.csv`

##  Tecnologías usadas
